# Student-Performance-Analysis

### 📌 Project Overview

#### <p>The objective of this project was to explore, clean, analyze, and visualize a student performance dataset using Python while following a standard data analysis workflow:<br> Load → Clean → Analyze → Visualize → Conclude.</p>

#### <p>This analysis focuses on understanding how various factors such as study time, gender, attendance, family background, and academic history influence students' final grades.</p>

---

### 🎯 Objectives

#### <li>Load and inspect the student performance dataset.</li>

#### <li>Perform basic data cleaning and validation.</li>

#### <li>Analyze student academic performance.</li>

#### <li>Answer key statistical and performance-related questions.</li>

#### <li>Create visualizations to identify trends and patterns.</li>

#### <li>Document the complete analysis process using Jupyter Notebook.</li>

---

#### <li>Jupyter Notebook</li>

### 📂 Dataset Information

#### <p>The project uses the Student Performance Dataset, which contains information about 395 students and 33 attributes related to demographics, family background, study habits, lifestyle factors, and academic performance.</p>

#### <p>The primary target variable analyzed in this project is <b>G3 (Final Grade)</b>, which represents the student's final academic score.</p>

---

### 🧹 Data Cleaning

#### <li>Loaded the dataset using Pandas.</li>

#### <li>Inspected dataset dimensions and data types.</li>

#### <li>Checked for missing values using <code>isnull()</code>.</li>

#### <li>Checked for duplicate records.</li>

#### <li>Created a cleaned dataset for analysis.</li>

#### <p>The dataset contained no missing values and was ready for analysis after validation.</p>

---

### 📊 Exploratory Data Analysis

#### The following questions were investigated:

#### <li>What is the average final grade (G3)?</li>

#### <li>How many students scored above 15?</li>

#### <li>Is study time correlated with academic performance?</li>

#### <li>Which gender performs better on average?</li>

---

#### <li>Scatter Plot: Absences vs Final Grade</li>

#### <li>Bar Chart: Average Grade by Study Time</li>

#### <p>These visualizations help identify trends, performance distributions, and relationships between different student attributes and academic outcomes.</p>

---

### 🔍 Key Insights

#### <li>Most students scored within the middle grade ranges.</li>

#### <li>Higher study time generally showed a positive relationship with final grades.</li>

#### <li>Attendance appeared to influence academic performance.</li>

#### <li>Students with fewer previous failures tended to achieve higher final grades.</li>

#### <li>Gender-based performance differences were analyzed through comparative statistics.</li>
